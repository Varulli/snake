# snake

A recreation of the video game Snake.

![snake-demo](https://github.com/user-attachments/assets/f89224d7-d7bd-4dd3-a80f-4f2e7e07d00f)

## Build Instructions

### Requirements
- Windows 7+
- C++ compiler
- CMake 3.5+

### Steps
```
git clone https://github.com/Varulli/snake.git
cd snake
mkdir build
cd build
cmake ..
cmake --build .
```

## Controls

Arrow Keys: Move snake

Enter: Quit
